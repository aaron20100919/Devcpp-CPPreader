# [Devcpp-CPPreader](https://github.com/aaron20100919/Devcpp-CPPreader.git)
用纯c++实现

### 快捷键说明:
1. F6 切换为`隐藏/显示`, 并且退回上一句
2. F7 上一句
3. F8 下一句
4. F9 退出

### 使用说明:
1. 参考 `LOOKME.mp4`;
2. 在 `DEV-CPP` 中编译 `test.cpp`;
3. 单击 `if` 语句前的行号;
4. 按 F5 调试;
5. 在**文件资源管理器中**运行 `read.exe`;
6. 默认小说是 `santi.txt`, 可以自己下载并另存为 `ANSI` 格式;
